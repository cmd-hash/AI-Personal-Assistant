**Project Description**

AI Personal Assistant Web App built using Flask and Mistral AI API.
This application allows users to interact with an AI assistant to ask questions and summarize emails in real-time through a simple and responsive web interface.

🚀 Features
Ask anything (AI-powered responses)
Email summarization (clear & concise output)
Real-time interaction (no page reload using JavaScript/AJAX)
Clean and minimal UI

🛠️ Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
API: Mistral AI
Environment Management: python-dotenv

⚙️ How It Works
User inputs a query or email in the UI
Frontend sends request to Flask backend
Backend calls Mistral API
Response is returned and displayed instantly
